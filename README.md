This is my thing where I keep things that I like while learning things.

## Look under the hood:
1. Clone/download repository to local folder.
2. Install dependencies using [npm](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm).
    ```
    npm install
    ```
3. Run a local development server at https://localhost:1234 using [Parcel](https://parceljs.org/getting_started.html).
    ```
    npx parcel src/index.html
    ```
